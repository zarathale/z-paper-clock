# z-paper-clock

Project repo for a model paper-clock build. The `source/` folder holds reference scans and transcriptions of an existing paper-clock book, kept here for personal reference. Original work derived from this study lives outside `source/` and will grow as the project develops.

## Layout

- `source/` — reference material (scans + transcriptions). Personal-use archive.
  - `scans-raw/` — original phone photos, renamed by page reference
  - `scans-clean/` — normalized, deskewed, cropped versions (to be produced)
  - `transcriptions/` — text of the preface, author's note, instructions, embedded labels
  - `inventory.md` — page-by-page log of what's in `scans-raw/`
- `work/` — derivative work, exploration, designs (to be populated)
