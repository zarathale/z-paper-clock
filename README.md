# z-paper-clock

Personal project repo for a model paper-clock build, studying *Make Your Own Working Paper Clock* by James Smith Rudolph (Harper & Row 1983; Perennial 2003 reprint). The `source/` folder holds reference scans and transcriptions of the book, kept here strictly for personal study. Anything published from this project should be original, derivative work — not a republication of the book.

## Layout

- `source/` — reference archive (personal use only)
  - `inventory.md` — page-by-page log mapping scan filenames to printed pages and content
  - `scans-raw/` — original phone photos, renamed to a `pNNN-short-description.jpg` convention so filenames sort in reading order
  - `scans-clean/` — dewarped, perspective-corrected, normalized to 10:13 portrait (2400×3120) or 20:13 landscape (4800×3120) for the two centerfold spreads
  - `transcriptions/` — markdown text of the book's prose and embedded-label content:
    - `covers-and-frontmatter.md` — front cover, title page (p. 1), copyright page (p. 2), back cover
    - `preface-clocks-asimov.md` — Isaac Asimov's introduction "Clocks" (pp. 3–4)
    - `authors-note.md` — James Smith Rudolph's author's note (p. 33)
    - `instructions.md` — full assembly instructions (pp. 34–40), including the construction sections (framework, mechanism, anchor and pendulum, hand mechanism, weight, face and case) and the adjustments section
    - `embedded-labels.md` — every label, axle assignment, direction marker, and figure caption printed *on* plates A–M, plus a piece-number-to-axle cross-reference
- `work/` — derivative design and exploration (to be populated)

## Status

Source documentation is complete: 27 cleaned scans cataloged in `inventory.md`, and full transcriptions of all prose and embedded-label content. The `work/` folder is reserved for original derivative work that grows out of this study.

## A note on scope

The book is held here for personal reference only. The book is in print and available from HarperCollins; if you want to build the clock, buy a copy. This repo exists to support a derivative paper-clock project, not to redistribute the source.
